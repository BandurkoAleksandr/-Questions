Task description

Вводится N слов. Можно ли их переупорядочить так, чтобы получилась "цепочка"
т.е. для каждого слова его первая буква должна совпадать с последней буквой предыдущего слова,
соответственно последняя буква последнего слова должна совпадать с первой буквой первого слова.
Дать ответ в виде "Можно"\"Нельзя".
Если такое упорядочение возможно, то вывести какую-нибудь цепочку слов.
